# [9. 回文数](https://leetcode-cn.com/problems/palindrome-number/)

## 1. 对半

一开始要检查到最后，看看两个数是不是完全一样，这样会溢出如果数字太大的话，看讨论区只检查一半就可以

```javascript
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function (x) {
    if (x < 0 || (x % 10 === 0 && x !== 0)) {
        return false;
    } 
    let val = 0;
    while (x > val) {
        val = val * 10 + x % 10;
        x = parseInt(x / 10);
    }
    return x === val || x === parseInt(val / 10) 
};
```
