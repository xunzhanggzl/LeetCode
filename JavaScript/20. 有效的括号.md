# [20. 有效的括号](https://leetcode-cn.com/problems/valid-parentheses/)

## 1. Stack

利用栈的原理，这里如果  `s.length`  是奇数，直接返回  `false`  提高效率

```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
const map = {
    "(": ")",
    "{": "}",
    "[": "]"
}
var isValid = function (s) {
    if (s.length % 2 !== 0) {
        return false;
    }   
    let stack = [];
    for (let i = 0; i < s.length; i++) {
        let ele = s[i];
        if (ele in map) {
            stack.push(map[ele]);
        } else {
            if (stack.pop() !== ele) {
                return false;
            }
        }
    }
    return stack.length === 0;
}
```
