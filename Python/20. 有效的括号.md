# [20. 有效的括号](https://leetcode-cn.com/problems/valid-parentheses/)

```python
class Solution(object):
    def isValid(self, s):
        if len(s) % 2 != 0:
            return False
        dic = {"{":"}","[":"]","(":")"}
        stack = []
        for char in s:
            if char in dic:
                stack.append(dic[char])
            else:
                if len(stack) == 0 or stack.pop() != char:
                    return False
        return len(stack) == 0
```
