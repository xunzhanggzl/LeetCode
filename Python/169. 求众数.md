# [169. 求众数](https://leetcode-cn.com/problems/majority-element/)

## 1. dic

最简单的方法

```python
class Solution(object):
    def majorityElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        dic = {}
        for num in nums:
            if num in dic:
                dic[num] += 1
            else:
                dic[num] = 1
            if dic[num] > len(nums)/2:
                return num
```

## 2. sort

```python
class Solution(object):
    def majorityElement(self, nums):
        nums.sort()
        return nums[len(nums) // 2]
```
