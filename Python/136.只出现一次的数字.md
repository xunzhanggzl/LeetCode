# [136. 只出现一次的数字](https://leetcode-cn.com/problems/single-number/)

## 1. dic

```python
class Solution(object):
    def singleNumber(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        dic = {}
        for num in nums:
            if num in dic:
                dic[num] += 1
            else:
                dic[num] = 1
        for prop in dic:
            if dic[prop] == 1:
                return prop
```

## 2. 异或 ^

```python
class Solution(object):
    def singleNumber(self, nums):
        res = 0
        for num in nums:
            res ^= num
        return res
```
