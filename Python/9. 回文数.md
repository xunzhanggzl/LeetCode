# [9. 回文数](https://leetcode-cn.com/problems/palindrome-number/)

```python
class Solution(object):
    def isPalindrome(self, x):
        """
        :type x: int
        :rtype: bool
        """
        if x < 0 or (x % 10 == 0 and x != 0):
            return False
        val = 0
        while x > val:
            val = val * 10 + x % 10
            x = x / 10
        return x == val or x == val / 10

```
